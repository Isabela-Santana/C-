# Descrição
*Descrição Técnica: > Desenvolvi um validador de cartões em C# utilizando o GitHub Copilot para a geração de expressões regulares (Regex) de alta precisão e implementação do Algoritmo de Luhn.

*Destaques do meu projeto:

    *Clean Code: Métodos com responsabilidade única e nomes semânticos.

    *Padrões de Projeto: Uso de Regex para extensibilidade de novas bandeiras.

    *Qualidade de Software: Implementação de testes unitários para garantir a confiabilidade da lógica de validação.

    *IA Integration: Documentação de como os prompts do Copilot ajudaram na refatoração do código.
